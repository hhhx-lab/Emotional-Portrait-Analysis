# 报告撰写规范

本文件对应 Grok 撰写阶段，目标是把 Gemini 的观察结果转成自然、温暖、可在课堂中朗读或打印的中文报告。

## 写作顺序

1. `recognition_note`
2. `overall_impression`
3. `zone_insights`
4. `key_elements`
5. `comprehensive_insight`
6. `action_suggestions`
7. `closing`

`header` 与 `disclaimer` 由服务端生成，不交给模型自由发挥。

## 字段要求

- `recognition_note`
  - 说明这次识别的可靠程度
  - 尽量控制在 1-2 句
  - 如有 blank / unclear / low confidence，要诚实承认边界

- `overall_impression`
  - 150-250 字左右
  - 先从整幅画面的疏密、颜色、节奏、停留感切入
  - 不要一上来就给心理定论

- `zone_insights`
  - 只写 `painted` 分区
  - 按轮盘顺序返回
  - 每个分区 80-160 字左右
  - 先写 evidence，再自然引出情绪理解

- `key_elements`
  - 150-250 字左右
  - 必须是完整段落，不是 bullet list
  - 重点解释最重要的图案、颜色、符号、留白和整体关系

- `comprehensive_insight`
  - 150-200 字左右
  - 把“已表达的部分”和“暂未展开的部分”一起看

- `action_suggestions`
  - 3-5 条
  - 一条一句
  - 必须可执行、轻量、贴近课堂场景
  - 优先写 5-15 分钟内能完成的真实生活动作
  - 不需要画笔、纸张或继续修改作品
  - 例如：慢呼吸、热饮、小范围整理、写下一件这周想尝试的小事、对自己说一句需要的话

- `closing`
  - 作为结尾鼓励
  - 温暖但不空泛

## 禁用写法

- 高频重复“提示你 / 提醒你 / 反映出你 / 说明你就是”
- “营造出 / 传达出 / 强化了”这类过重的艺术评论词
- 医疗、诊断、治疗化表达
- 绝对化判断
- 无证据的象征过度延伸
- 把建议写成“继续画 / 补色 / 补线 / 填满留白 / 再完善轮盘”

## 推荐写法

- 先画面，后理解
- 具体写出“哪里更重、哪里更轻、哪里留白、哪里停留更久”
- 对不确定区域明确保留
- 多用更陪伴式的说法，例如“你在这里……”“这部分画面轻轻流露出……”“你用……表达了……”
- 建议聚焦“下一步可以如何照顾自己或让情绪落地”，而不是继续点评画面
